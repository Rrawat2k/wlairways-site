# White Line Airways — Official Website

Express Logistics Solutions Since 1993

## Tech Stack
- Pure HTML/CSS/JS — single file SPA
- [Supabase](https://supabase.com) — database for blog posts & leads
- [Web3Forms](https://web3forms.com) — form email notifications
- Hosted on [GitHub Pages](https://pages.github.com)

## Live Site
[www.wlairways.com](https://www.wlairways.com)

## Admin Dashboard
Click **Login** on the site nav.  
Default credentials: `admin` / `wla@2024`

## Features
- 6-page SPA (Home, About, Services, Blog, Contact, Get a Quote)
- Blog with Supabase-powered CMS
- Admin dashboard — publish posts, view leads, SEO checklist
- Form submissions saved to Supabase + emailed via Web3Forms
- Full SEO: Schema.org, OG tags, dynamic meta per page
- Sitemap & robots.txt generators

## Updating the Site
1. Edit `index.html` locally
2. Open GitHub Desktop → Commit → Push
3. Site updates automatically within 1–2 minutes
